系统操作说明：
（操作前，请修改配置代码，即修改neo4j_test中的用户名及密码
即修改graph = Graph('http://localhost:7474', auth=("neo4j", "neo4j"))）

首先在neo4j-community的bin目录下执行neo4j.bat console，启动数据库
清空数据库 match (n) detach delete n
-->
执行proxy_test.py,测试代理，将本地网络ip添加到芝麻代理白名单,如果不用代理，可能出现情感分析
失败的情况，或将情感分析代码部分的get_proxy()方法注释掉
-->
运行src目录下的douban.py
-->
点击登录或注册后登录，注册需按要求注册填写
-->
需先点击"加载豆瓣电影信息"按钮,等待程序进行，在console中将输出爬取进度，当出现“100%”
后方可再次点击其他按钮，否则将产生错误或数据不完整
-->
点击“导出数据构建知识图谱”,生成excel数据文件与本地知识图谱数据库,查看数据库
-->
点击获取词频，可根据想了解的词频统计信息进行选择，将生成相应的词频图表
-->
点击获取情感分析，在输入框输入所要情感分析的电影名称，点击搜索或回车，将在界面上看到
10条热门评论内容及情感分析得分，并显示用户情感分析得分折线图，情感程度越接近1，表示积极
越趋近0，表示消极
-->
返回登录界面或退出系统
